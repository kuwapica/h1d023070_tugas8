## Penjelasan kode 
project ini memilik dua folder yang aktif digunakan pada folder `lib` yaitu folder `model` dan `ui` (yang berisi tampilan dari aplikasinya). di dalam folder `model` terdapat 3 file yaitu: `login.dart`, `registrasi.dart`, serta `produk.dart`. adapun folder `ui` berisi 5 file yaitu: `registrasi_page.dart`, `login_page.dart`, `produk_page.dart`, `produk_detail.dart`, dan `produk_form.dart`. serta terdapat file yg aktif digunakan yaitu `main.dart`. pada file `main.dart` menngimpor library diantaranya:
```
import 'package:flutter/material.dart';
import 'package:tokokita/ui/registrasi_page.dart';
```
begitu pula file file lain di folder `ui` mengimpor hal yang sama.
